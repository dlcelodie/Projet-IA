The concept

FashionAI is an artificial intelligence based fashion assistant. It relies on Computer Vision and Machine Learning to provide 2 main functionalities which are similar clothes retrieval and image based outfit recommendation. It can be integrated to marketplaces in order to help customers find the ideal outfit.

How it works: Similar clothes retrieval

The similar clothes retrieval component is based on 2 main steps. The first one is a category clothes classification, this step's main goal is to focus our research on the specific product. It's based on a pretrained classification Convolutional Neural Network (CNN), we use transfer learning to adapt it to our task. In order to create the model we used Teachablemachine and the MMFashion pretrained models/source code. For the dataset we used the MMFashion attribute prediction dataset. The second step consist of a similarity based research using a pretrained CNN from the MMFashion repository giving a similarity score between 2 images. For this step we use the In shop retrieval dataset.

How it works: Image based outfit recommendation

The outfit recommendation is based on a pretrained CNN from the MMFashion repository, trained on the Deep Fashion dataset. The CNN model gives a similarity score between the given image and images from the different categories. Then we combine the similar clothes to create an harmonious combination. The outfits are then recommended to the customers based on that score.

MMFashion model exploitation

In order to use the MMFasion open source pretrained models we used the ONNX tool for model weights translation. With this tool, we've been able to translate the pretrained weights from pytorch native weight files to tensorflow ones.

Website architecture

In order to make accessible our 2 main functionalities we imagined the following architecture composed from 2 main parts: the client side which is basically a browser, and the server side (NodeJS) containing our image processing, models, and source code.

Our competitors

some of our main competitors we've found in the market

Watiz.io

Watiz implements the similar clothes retrieval functionality. It enable customers to search for similar clothes from thousands of references in their catalogue.

Stylesnap

Amazon stylesnap is an image based clothes search engine which gives similar products, to a given image, in the amazon website.

Screenshop

Screenshop is another application that does image similarity. It gives similar products to the given image and where to buy them.

Conclusion

In this project we've been able to discover how to use Artificial Intelligence to the fashion field. We learned how we can apply Computer Vision, Machine Learning, Supervised & Transfer learning and other AI concepts & tools. Our solution has a great potential and can change the way people buy products. In fact, it improves the user's experience and enable's them to gain time and get accurate recommendations.
